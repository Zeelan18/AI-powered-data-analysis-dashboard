# 📊 AI-Powered Data Analysis Dashboard

## 🚀 Overview

An intelligent analytics platform built using Python, Streamlit, Machine Learning, and AI integration.

This dashboard enables users to upload CSV datasets and perform:

* Advanced data visualization
* Data quality analysis
* Correlation analysis
* AI-powered insights
* Machine learning predictions
* PDF report generation

---

## ✨ Features

* 📂 CSV Dataset Upload
* 📈 Interactive Visualizations
* 🔥 Correlation Heatmaps
* 🛡 Data Quality Analysis
* 🤖 AI Analytics Assistant
* 🧠 Machine Learning Predictions
* 📑 PDF Report Generation
* ⬇ CSV Export
* 🎨 Modern SaaS UI

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Scikit-learn
* OpenAI/OpenRouter API
* ReportLab

---

pip install -r requirements.txt
